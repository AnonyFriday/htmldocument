    video, source

    Video rendering supplied by those format standard:

    • 3GP:

      • Video format on memory limited storage capacity mobile phone (3G phone)

    • MP4:

      • Video format on wide range of devices, including computer, smartphones, and tablets

    • WebM:

      • open-source video file format developed by Google

      • high-quality video compression while keeping file sizes relatively small.

    • OSG:

      • open-source video file format developed by Google

      • Like WebM, OGV is optimized for online video playback.
